Install
------------------------------------------------------------------------------

``acore_server_config`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install acore-server-config

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade acore-server-config
